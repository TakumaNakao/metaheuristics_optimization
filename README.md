## What is this?
Demonstration of meta-heuristics algorithms.

## Dependencies
* CMake
* Eigen3
* Matplot++

## Docker
Launch Docker with the following command:
```
bash docker_bringup.sh
```

## Build
build
```
bash build.sh
```
clean build
```
bash clean_build.sh
```
